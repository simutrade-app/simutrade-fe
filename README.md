## 📘 Project Overview
SimuTrade is an AI-powered Trade Supply Chain Simulation designed to deliver a seamless and interactive user experience.

SimuTrade is deployed on [simutrade.app](https://simutrade.app) and it's API (Back End) deployed on [api.simutrade.app](https://api.simutrade.app)

SimuTrade Frontend Repository: [https://github.com/simutrade-app/simutrade-fe](https://github.com/simutrade-app/simutrade-fe)<br>
SimuTrade API Backend Repository: [https://github.com/simutrade-app/simutrade-api](https://github.com/simutrade-app/simutrade-api)<br>
SimuTrade RAG + AI Backend Repository: [https://github.com/simutrade-app/simutrade-ai](https://github.com/simutrade-app/simutrade-ai)

## 🛠️ Tech Stacks

- **React**: A popular JavaScript library for building user interfaces using a component-based architecture.
- **Vite**: A lightning-fast frontend build tool and development server for modern web projects.
- **TypeScript**: A typed superset of JavaScript that enhances code quality and developer experience with static type checking.
- **Tailwind CSS**: A utility-first CSS framework for rapidly building custom designs without leaving your markup.
- **Radix UI**: A low-level UI component library offering unstyled and accessible primitives for building high-quality design systems.
- **Ant Design (antd)**: A comprehensive React UI framework offering polished components and design resources out of the box.
- **React Hook Form**: A performant form library for React that simplifies form validation and state management.
- **Zod**: A TypeScript-first schema validation library for parsing and validating data.
- **React Router DOM**: A standard routing library for React applications enabling navigation between components and views.
- **React Query (@tanstack/react-query)**: A powerful tool for fetching, caching, and updating server state in React applications.
- **Socket.IO Client**: A real-time bidirectional event-based communication library for client-server messaging.
- **Axios**: A promise-based HTTP client for making requests to external APIs from the browser.
- **Recharts**: A composable charting library built on React and D3 for visualizing complex data.
- **Leaflet + React Leaflet**: A leading open-source library for interactive maps, extended with React bindings.
- **Lucide React**: A set of beautiful and consistent open-source icons for React.
- **Font Awesome + React FontAwesome**: A popular icon library with React components for scalable vector icons.
- **GSAP (GreenSock Animation Platform)**: A robust animation library for high-performance and complex animations.
- **Embla Carousel**: A lightweight, extensible carousel library for touch-friendly slide interfaces.
- **Prettier**: An opinionated code formatter that enforces consistent code style across your project.
- **ESLint**: A static code analysis tool to find and fix problems in JavaScript and TypeScript codebases.
- **PostCSS**: A tool for transforming CSS with JavaScript plugins like autoprefixer.
- **Vite Plugin React**: A plugin that provides React fast refresh and JSX support in Vite.
- **Tailwind Typography Plugin**: A plugin that adds beautiful typographic defaults to Tailwind CSS.
- **clsx / class-variance-authority**: Utilities for conditionally joining classNames in JSX and managing variant-based styles.
- **Vaul**: A component abstraction over Radix Dialog for creating accessible modals with better ergonomics.
- **Sonner**: A minimal toast notification library for React with built-in accessibility and animation support.
- **date-fns**: A lightweight library for modern date manipulation using pure functions.
- **React Markdown**: A component for rendering Markdown as React elements.
- **React Resizable Panels**: A flexible layout library for building draggable and resizable panels.

## ⚙️ Installation
To set up SimuTrade API backend, start by clone the repository
   ```sh
   git clone https://github.com/simutrade-app/simutrade-fe.git
   cd simutrade-api
   ```

To set up the Frontend locally, run:
   ```sh
   npm run dev
   ```
